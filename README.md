# my utility function
よく使うような機能をあらかじめ関数化しておく。
最悪忙しい時に fork したり、コピペしたりできるように...
そのうち、privateのnpmモジュールにしたいと思っている。

### environment
---
- node v14.16.0
- vite ^2.5.1
- typescript ^4.3.2

### Hoe To
---
- clone 
```bash
git clone git@github.com:tsuchiya-kai/my-utility-function.git
```

- 初回install
```bash 
yarn 
```

- local
```
yarn dev
```

- production build
```
yarn build
```
